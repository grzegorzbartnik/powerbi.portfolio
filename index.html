Management Dashboard

Description

The first 3-pages dashboard shows an overwiev of the costs and profits of individual projects with theirs types over a period of 5 years (2021-2025).

First page contains the most important KPIs and donut charts with distribution of costs and profits in four regions. The last page shows a table with a detailed summary of the projects.

The left side of the dashboard contains 2 slicers that allow to select projects or their types.

Sales Performance Dashboard

Desctription

Second dashboard - more technically advanced than the first - focuses mainly on showing PY vs CY performance (Sales, Profit, Margin, Orders and ATV). Measures in DAX language were created to present this comparisons.

On the left side of the dashboard there is a dynamic open/hide slicer panel with filters. This solution allows to save space for key visualizations.

Thermos CO. Analysis Dashboard

Desctription

The third dashboard is also a multi-page dashboard. Navigation panel on the left allows to freely switch between the pages.

Dashboard shows a comparison of the sales over several years of a global thermos manufacturer. DAX measures allowed to create PY vs CY performance.

Summary

The main goal of my portfolio was to showcase my development in creating dashboards. In this summary I want to present what skills I acquired during the process of creating dashboards.

General skills: loading data into Power BI from different sources (Excel, CSV), perform ETL process using Power Query, creating relationships between tables and modeling a star schema, publishing dashboards on Power BI Service.

Visualization skills: creating measures with the DAX language, projecting multi-page dashboards with dynamic open/hide page navigation panel, using buttons for example to open/hide certain objects on the page, creating creative shapes on a specific theme.

In addition, creating and using elements like: conditional formatting, bookmarks, custom tooltips and grouping objects.
